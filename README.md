# Topic Generator

A simple app that gives you random conversation topics.<br/>
ランダムに話題を考えてくれるアプリです。

## DEMO

https://masayasanyo.github.io/topic_generator/

## Features

-Provides random topics.<br/>
話題をランダムに提供<br/>
-Bilingual support in English and Japanese.<br/>
英語、日本語の２ヶ国語対応<br/>

## Tech Stack
  
-HTML, CSS, Javascript
